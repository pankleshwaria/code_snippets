# code_snippets
Python tutorial code snippets

## Quick Start
Installation steps if you want to try it out
```bash
git clone https://github.com/pankleshwaria/code_snippets.git
cd <tutorial-folder>
run python <tutorial-file.py>
```

## Requirements
* Python 3.4+
